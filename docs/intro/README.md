# Вступ

Протокол HTTP може використовуватись не лише для веб-сторінок. Це ще й потужна платформа для створення API, які надають послуги та дані.
Протокол HTTP простий, гнучкий і поширений. Практично будь-яка існуюча платформа має бібліотеку HTTP, тому служби HTTP доступні у 
великій кількості клієнтів, включаючи браузери, мобільні пристрої та традиційні офісні програми.
